# USA–IRAN Conflict 2026 NLP Analysis

This repository contains the dataset and all implementation code used to analyze social media discussions about the **USA–Iran conflict (2026)**.

The project includes different Natural Language Processing (NLP) steps such as data preprocessing, sentiment labeling, topic modeling, and model training.

## Contents

This repository includes:

- Dataset used for the analysis
- Data preprocessing scripts
- Sentiment labeling process
- Topic modeling implementation
- Model training code
- Other supporting scripts used during the research

## Workflow

The general workflow of the project is:

1. Data Collection  
2. Text Preprocessing  
3. Sentiment Labeling  
4. Topic Modeling  
5. Model Training  
6. Evaluation and analysis
